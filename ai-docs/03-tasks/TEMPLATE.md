## Что нужно сделать

Конкретный список шагов. Не «подумать, как лучше» — архитектор уже подумал.

1. Создать файл `backend/apps/directory/services/panel_mover.py`
2. Класс `PanelMover`:
   ```python
   class PanelMover:
       def __init__(self, activity_logger): ...

       def move(
           self,
           panel: Panel,
           to_department: Department,
           actor: User,
           comment: str = ''
       ) -> Panel:
           # проверки
           # перемещение
           # лог
           # уведомление
   ```
3. Валидация:
   - Нельзя перемещать в ЗИП панель с активной заявкой (исключение `PanelHasActiveApplicationError`)
   - Нельзя перемещать в `monitor` (установку в экран) напрямую — только через `replace_panel_in_cell`
4. Успешное перемещение:
   - `panel.department = to_department`; `panel.save(update_fields=['department', 'updated_at'])`
   - `activity_logger.log(actor=actor, target=panel, event_type='panel.moved', description=..., payload={...})`
   - `transaction.on_commit(lambda: notification_queue.enqueue(...))`
