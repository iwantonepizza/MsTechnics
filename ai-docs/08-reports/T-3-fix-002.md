## Что проверено

- `apps/interface/api/v1/applications/views.py`:
  - `destroy()` разрешает удаление только для статуса `sent_to_control`;
  - окно удаления ограничено 5 минутами;
  - используется whitelist `is_creator or is_admin`;
  - `creator is None` больше не пропускает non-admin пользователя;
  - `ActivityLog` пишется перед `app.delete()`.
- `apps/interface/api/v1/auth/views.py`:
  - `RefreshView` читает refresh из httpOnly cookie;
  - старый refresh вызывает `old_refresh.blacklist()`;
  - создаётся новая пара refresh/access;
  - новый refresh возвращается в cookie.
- `apps/interface/tests/test_security.py` содержит regression tests для B3/B4.
