## Что сделано

- `TransitionModal` работает через общий config `transitionConfigs.ts`.
- Поддержаны основные application transitions: `apply_in_control`, `sent_to_service`, `work_in_service`, `done`, `unable`, `archive_done`, `archive_unable`, `delete_application`.
- Для `sent_to_service` добавлен обязательный dropdown исполнителя.
- Добавлен frontend hook `useExecutors()` и backend endpoint `/api/v1/executors/`.
- Backend transition теперь не только валидирует `executor_id`, но и реально назначает исполнителя через `application_service.set_executor`.
- OpenAPI schema и generated TypeScript schema обновлены.
