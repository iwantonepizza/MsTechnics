## Архитектурные Решения

- Сканер сделан без `chardet`/`charset-normalizer`; используется жёсткая проверка UTF-8 и список характерных mojibake-маркеров.
- Hook запускается как `language: system`, чтобы использовать уже установленный Python в рабочем окружении и не плодить отдельную runtime-зависимость.
- Проверка по умолчанию идёт по `git ls-files`, а в pre-commit принимает список staged filenames.
