## Архитектурные решения

- Тема вынесена в один shared-контур `ThemeProvider`, а не размазана по страницам: это исключает расхождение между header toggle, profile radio-group и системной темой.
- Для режима `system` `<html>` остаётся без `data-theme`, а значения берутся из `@media (prefers-color-scheme: dark)`, как и требовалось в задаче.
- `tokens.css` оставляет совместимые алиасы (`--fg`, `--accent`, `--warn`, `--err`, `--ok`, и т.д.), чтобы Phase 7 можно было внедрить без массового переписывания уже существующих компонентов.
