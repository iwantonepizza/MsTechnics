# T-8-011 — 4 кнопки экрана: схема / проект / контакты / фото

## Что сделано

- В `apps/interface/api/v1/displays/serializers.py` `GET /displays/<slug>/` теперь отдаёт `contacts` и `photos` вместе с `file_url` и `project_photo_url`.
- В `apps/interface/api/v1/displays/views.py` маршрут `photos` доведён до `GET / POST / DELETE`, с изменением фото только для `service/admin`.
- В `frontend/src/shared/api/types.ts` добавлены типы `DisplayContact` и `DisplayPhoto`.
- В `frontend/src/pages/department/DepartmentListPage.tsx` добавлены 4 action-кнопки на карточке экрана:
  - просмотр электросхемы
  - просмотр проекта
  - список контактов с `tel:` и копированием номера
  - фотобанк с загрузкой и удалением без перезагрузки страницы

## Что важно по продукту

- Контакты в SPA только читаются; редактирование остаётся в Django admin.
- Фотобанк редактируется из сервисного интерфейса, что соответствует решению владельца для `service/admin`.
